# Task-Tracker# 📋 Task Tracker Web App

A browser-based **Task Tracker** built using **HTML, CSS, JavaScript, and TypeScript**.

This project was developed as part of an internship assignment to demonstrate front-end development skills including semantic HTML, DOM manipulation, TypeScript, Local Storage, classes, and generics.

---

## 📂 Project Structure

```
task-tracker/
├── index.html
├── style.css
├── main.js
├── tasks.ts
├── tasks.js
├── tsconfig.json
└── README.md
```

---

## 🚀 Features

- Add new tasks
- Set task priority (Low / Medium / High)
- Choose a due date
- Mark tasks as Done or Pending
- Delete individual tasks
- Clear all tasks
- Filter tasks (All / Pending / Done)
- Sort tasks by Priority or Due Date
- Highlight overdue tasks
- Live task counter
- Summary table
- Priority summary using a generic `groupBy()` function
- Save tasks using Local Storage
- Restore tasks after page refresh

## 📖 How to Run

### Clone the Repository

```bash
git clone https://github.com/your-username/task-tracker.git
```

### Open Project

```bash
cd task-tracker
```

### Compile TypeScript

```bash
tsc
```

This generates:

```
tasks.js
```

### Run

Open **index.html** in your browser.
